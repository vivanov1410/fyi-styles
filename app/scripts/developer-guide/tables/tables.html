Tables

Basic

For basic styling — border, stripes and light padding — add the base class .table to any <table>. It may seem super redundant, but given the widespread use of tables for other plugins like calendars and date pickers, we've opted to isolate our custom table styles.

Contextual

Use contextual classes to color table rows or individual cells.

Class Description
.is-active Applies the hover color to a particular row or cell
.is-success Indicates a successful or positive action
.is-info Indicates a neutral informative change or action
.is-warning Indicates a warning that might need attention
.is-danger Indicates a dangerous or potentially negative action

Responsive

Create responsive tables by wrapping any .table in .table--responsive to make them scroll horizontally on small devices (under 768px). When viewing on anything larger than 768px wide, you will not see any difference in these tables.
